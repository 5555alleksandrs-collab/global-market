# Telegram Planner Bot

Личный Telegram-планер без OpenAI и без GPT: задачи, дедлайны, напоминания и команды для быстрого управления.

## Что умеет

- хранит задачи отдельно для каждого чата
- сохраняет данные в `data/tasks.json`
- присылает напоминания по времени
- показывает список дел и задачи на сегодня
- поддерживает быстрое добавление задач прямо обычным сообщением
- понимает простые фразы вроде `завтра 19:00 купить молоко`
- понимает короткие текстовые команды вроде `сегодня`, `список`, `готово 3`
- показывает постоянную клавиатуру с быстрыми действиями

## Команды

- `/start` - краткая инструкция
- `/help` - список команд
- `/add Купить молоко`
- `/add Купить молоко | 2026-03-18 19:00`
- `/add Купить молоко | 2026-03-18 19:00 | 2026-03-18 18:00`
- `/list` - все активные задачи
- `/today` - просроченные и сегодняшние дедлайны
- `/done 3` - отметить задачу выполненной
- `/delete 3` - удалить задачу
- `/deadline 3 2026-03-18 19:00` - поставить дедлайн
- `/remind 3 2026-03-18 18:00` - поставить напоминание

Можно просто отправить текст без команды, и бот добавит его как новую задачу.

## Быстрый режим

Можно писать почти по-человечески:

- `завтра 19:00 купить молоко`
- `купить молоко в 19:00`
- `напомни завтра 18:00 купить молоко`
- `сегодня`
- `список`
- `готово 3`

## Формат даты

Используй формат `YYYY-MM-DD HH:mm`.

Пример:

```text
/add Подготовить отчет | 2026-03-18 19:00 | 2026-03-18 17:30
```

## Запуск

1. Установите зависимости:

   ```bash
   npm install
   ```

2. Создайте `.env`:

   ```bash
   cp .env.example .env
   ```

3. Укажите токен Telegram-бота:

   ```env
   TELEGRAM_BOT_TOKEN=your_telegram_bot_token_here
   ```

4. Запустите бота:

   ```bash
   npm run dev
   ```

## Как получить Telegram-токен

1. Откройте `@BotFather` в Telegram
2. Выполните команду `/newbot`
3. Задайте имя и username бота
4. Скопируйте токен и вставьте его в `.env`

## Хранение данных

- все задачи лежат в `data/tasks.json`
- данные сохраняются после каждого изменения
- после перезапуска бот продолжает работать с тем же списком задач
