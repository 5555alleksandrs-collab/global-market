# WB SuperBot

Telegram-бот для автоматического бронирования поставок на Wildberries.

## Возможности

- **Авто-бронь** — бот сам ищет свободные слоты и бронирует поставку по вашим параметрам (склад, тип, период, коэффициент)
- **Поиск таймслотов** — уведомление при появлении подходящего слота (без авто-брони)
- **Перераспределение остатков** — создание заявок на перемещение товара между складами WB
- **Статистика** — история бронирований по складам и датам
- **Оплата** — поддержка YooKassa + ручное пополнение через команды администратора

## Быстрый старт

### 1. Требования

- Python 3.11+
- Токен Telegram-бота ([@BotFather](https://t.me/BotFather))
- API токен WB продавца (Личный кабинет → Настройки → Доступ к API)

### 2. Установка

```bash
# Клонируйте / скопируйте папку проекта
cd wb_superbot

# Создайте виртуальное окружение
python3 -m venv venv
source venv/bin/activate        # Mac/Linux
# venv\Scripts\activate         # Windows

# Установите зависимости
pip install -r requirements.txt
```

### 3. Конфигурация

```bash
cp .env.example .env
nano .env   # или откройте любым редактором
```

Заполните обязательные поля:
```
BOT_TOKEN=<токен от @BotFather>
WB_API_TOKEN=<токен из кабинета WB Партнёры>
ADMIN_IDS=<ваш Telegram ID>
```

### 4. Запуск

```bash
python main.py
```

Бот запустится, создаст базу данных `wb_bot.db` и начнёт polling.

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Главное меню |
| `/settoken` | Задать токен WB продавца |
| `/cancel` | Отменить текущее действие |

### Команды администратора

| Команда | Описание |
|---------|----------|
| `/addabron <user_id> <count>` | Начислить авто-брони пользователю |
| `/addsub <user_id> <days>` | Активировать подписку на N дней |

## Как создать авто-бронь (пошагово)

1. Нажмите **⚡ Авто-бронь**
2. Выберите **склад** WB
3. Выберите **тип поставки** (Короб / Монопаллета / Суперсейф)
4. Выберите **период** поиска (неделя / месяц / свои даты)
5. Выберите **максимальный коэффициент** (х0 = бесплатно)
6. Выберите **защиту** (рекомендуется включить — не бронирует ближайшие 72ч)
7. Введите **ID черновика** поставки из ЛК WB

Бот начнёт опрашивать API WB каждые `POLL_INTERVAL` секунд.
Как только появится подходящий слот — автоматически забронирует и пришлёт уведомление.

### Как создать черновик в ЛК WB

1. Перейдите в **ЛК WB Партнёры → Поставки → + Новая поставка**
2. Добавьте артикулы и укажите количество
3. **НЕ выбирайте склад и дату** — просто кликните в левое меню
4. Черновик появится во вкладке **Черновики**
5. Скопируйте его ID (число в URL или в списке)

## Оплата

### Без YooKassa (рекомендуется для начала)

Установите `PAYMENTS_ENABLED=false` в `.env`.
Пополнение вручную через команды администратора:
```
/addabron 123456789 10    # добавить 10 авто-броней
/addsub 123456789 30      # подписка на 30 дней
```

### С YooKassa

1. Зарегистрируйтесь на [yookassa.ru](https://yookassa.ru)
2. Получите `shop_id` и `secret_key`
3. Заполните в `.env`:
```
PAYMENTS_ENABLED=true
YOOKASSA_SHOP_ID=123456
YOOKASSA_SECRET_KEY=test_xxx
```

## Важные замечания

- **POLL_INTERVAL** не ставьте меньше 30 секунд — риск временной блокировки со стороны WB
- Токен WB хранится в базе данных SQLite локально, никуда не передаётся
- Бот использует **официальный WB Seller API** (`supplies-api.wildberries.ru`)
- Перераспределение остатков доступно не всем продавцам — нужна квота от WB
